What is Kotlin?

Kotlin คืออะไร

Kotlin เป็นภาษาโปรแกรมที่มีการพิมพ์แบบ static โดยออกแบบมาเพื่อให้สามารถแสดงผลได้อย่างชัดเจน ปลอดภัย และสามารถทำงานร่วมกับ Java ได้อย่างดี ทำให้เป็นทางเลือกยอดนิยมสำหรับการพัฒนาแอปพลิเคชันบนมือถือและเว็บไซต์ มีฟีเจอร์ภาษาที่ทันสมัยและเป็น Open-Source

Pros and Cons.

ข้อดี ข้อเสีย

ข้อดี
1. ทำงานร่วมกันได้กับ Java
2. มีคอมมูนิตี้ที่ใหญ่
3. มี null safety และ type inference ที่ช่วยลดความผิดพลาดในตอนรัน
ข้อเสีย
1. ไม่มีนิยมมากเท่ากับภาษาที่มีอยู่แล้ว เช่น Java หรือ Python
2. การ support อาจไม่มากพอในหลายอย่าง

Famous application that are built with Kotlin

แอปพลิเคชันดังที่ถูกสร้างด้วย Kotlin

Pinterest เป็นแพลตฟอร์มสื่อสังคมที่ได้รับความนิยมสำหรับการแบ่งปันไอเดียและแรงบันดาลใจ แอป Pinterest เริ่มต้นพัฒนาด้วยภาษา Java แต่ทีมพัฒนาภายหลังใน version Android ได้ย้ายมาใช้ Kotlin เนื่องจากมีฟีเจอร์ภาษาที่ทันสมัย syntax สั้นและกระชับ และเพิ่มประสิทธิภาพในการพัฒนา
