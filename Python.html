What is Python?

Python คืออะไร

Python เป็นภาษาโปรแกรมระดับสูงที่ใช้งานกันอย่างแพร่หลาย โดยมีจุดเด่นคือสัญลักษณ์ซินเท็กซ์ที่เข้าใจง่ายและชุมชนที่ใหญ่กว่าในการพัฒนา มีการนำไปใช้ในหลายสาขา เช่นการพัฒนาเว็บไซต์ การวิเคราะห์ข้อมูล และ AI อื่นๆ นอกจากนี้ยังมีคุณสมบัติในการใช้งานฟรี โอเพนซอร์ส และรองรับการทำงานบนหลายแพลตฟอร์ม

Pros and Cons.

ข้อดี ข้อเสีย

ข้อดี
1. Python มีการเขียนที่เข้าใจง่าย เหมาะสำหรับการเริ่มต้น
2. มีคอมมูนิตี้ที่ใหญ่
3. มีความอเนกประสงค์
ข้อเสีย
1. ค่อนข้างช้าเพราะต้องเอาไป interpreted ก่อน
2. ใช้ Memory มากทำให้ใช้ใน mobile platform ไม่ค่อยดี
