What is JavaScript?

JavaScript คืออะไร

จาวาสคริปต์ (JavaScript) เป็นภาษาทีมีลักษณะการเขียนแบบโพรโทไทป์ (Prototyped-based Programming) ส่วนมากใช้ในหน้าเว็บเพื่อประมวลผลข้อมูลที่ฝั่งของผู้ใช้งาน และก็ยังมีใช้เพื่อเพิ่มเติมความสามารถในการเขียนสคริปต์โดยฝังอยู่ในโปรแกรมอื่น ๆ เป็นภาษาที่มีการใช้งานกว้างขวางและหลากหลายมาก

Why should you learn JavaScript?

ทำไมคุณควรเรียน JavaScript

จาวาสคริปต์ (JavaScript) เป็น 1 ในภาษาที่ผู้พัฒนาเว็บทุกคนต้องเรียน
1. HTML เพื่อจำกัดความเนื้อหาของเว็บเพจ
2. CSS เพื่อระบุ layout ของเว็บ
3. JavaScript เพื่อกำหนด(program)การกระทำของเว็บ

Famous Apps Built with JavaScript.

แอพดังที่สร้างด้วย Javascript
