What is C#?

C# คืออะไร

C# เป็นภาษาโปรแกรมมิ่งที่ใช้งานง่าย ถูกพัฒนาโดย Microsoft สำหรับการสร้างแอปพลิเคชันเดสก์ท็อป แอปพลิเคชันเว็บ และเกม มันเป็นที่นิยมสำหรับการพัฒนา .NET และมีคุณสมบัติที่มีประโยชน์มากมาย เช่น garbage collection และ type safety

Pros and Cons.

ข้อดี ข้อเสีย

ข้อดี
1. เรียนรู้ง่าย
2. syntax คล้ายภาษาหลายๆภาษา
3. มี Garbage Collection
ข้อเสีย
1. จำกัดกับ Microsoft
2.อาจยากต่อการเขียนโค้ดระดับต่ำ เมื่อเปรียบเทียบกับภาษาอื่นๆ เช่น C

Famous game engine that uses C#

เกมเอนจิ้นดังที่ใช้ C#

ยูนิตี้ (Unity) ใช้ C# เป็นภาษาโปรแกรมมิ่งหลักโดยส่วนใหญ่ แต่ยูนิตี้ยังรองรับภาษาโปรแกรมอื่นๆ เช่น UnityScript, Boo, และ HLSL ด้วย
