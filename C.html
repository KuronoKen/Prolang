What is C?

C คืออะไร

C เป็นภาษาโปรแกรมมิ่งระดับสูงที่นิยมและมีประสิทธิภาพสูง มักถูกนำมาใช้สำหรับการโปรแกรมระบบและระบบฝังตัว มี community ที่ใหญ่โดยเป็นที่รู้จักกันดีเนื่องจากความสามารถในการ port กับอย่างอื่นสูงและความสามารถในการจัดการหน่วยความจำระดับต่ำได้ง่าย ๆ

Pros and Cons.

ข้อดี ข้อเสีย

ข้อดี
1. มีประสิทธิภาพสูง
2. สามารถคอมไพล์และรันบนแพลตฟอร์มต่างๆได้
3. ควบคุมหน่วยความจำระดับต่ำได้
4. มีคอมมูนิตี้ขนาดใหญ่
ข้อเสีย
1. ไม่มีการสนับสนุนสำหรับออบเจกต์หรือคลาส

Famous Software that are built using C

แอปพลิเคชันดังที่ถูกสร้างด้วย C

ภาษา C ถูกนำมาใช้ในแอปพลิเคชันซอฟต์แวร์หลายตัวที่ได้รับความนิยมอย่างมาก เช่นระบบปฏิบัติการเช่น Windows, Linux, และ macOS รวมถึงระบบฝังตัวในอุตสาหกรรมต่าง ๆ เช่นการบินและอุตสาหกรรมยานยนต์และอุปกรณ์การแพทย์ นอกจากนี้ C ยังถูกนำมาใช้ในเครื่องมือพัฒนาเกมหลายตัว เช่น Unreal Engine และ Unity
