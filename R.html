What is R?

R คืออะไร

R เป็นภาษาโปรแกรมที่นิยมใช้สำหรับการคำนวณทางสถิติและกราฟิก มีความสามารถในการวิเคราะห์ข้อมูล การเรียนรู้ของเครื่อง และการแสดงผลข้อมูลอย่างหลากหลาย นอกจากนี้ R ยังมีชุมชนผู้ใช้และนักพัฒนาที่ใหญ่และกระแสความคิดสร้างสรรค์ที่ต่อเนื่อง

Pros and Cons.

ข้อดี ข้อเสีย

ข้อดี
1. เหมาะกับการวิเคราะห์และแสดงผลสถิติ
2. คอมมูนิตี้ที่ใหญ่ และ เป็น open-source
ข้อเสีย
1. ใช้ memory มากจากการใช้ในงานที่ข้อมูลมาก
2. มีความยากในการเรียนรู้
3. Gui ไม่ค่อยมีความ appealing

Statistic Software built with R

ซอฟต์แวร์สถิติที่สร้างด้วย R

RStudio เป็นแอปพลิเคชันฟรีและโอเพนซอร์ซสำหรับ Integrated Development Environment (IDE) สำหรับภาษาโปรแกรม R มันให้ส่วนติดต่อผู้ใช้งานสำหรับเขียนและรันโค้ด R อย่างเป็นมิตรต่อผู้ใช้งาน รวมถึงมีคุณสมบัติสำหรับการจัดการและแสดงข้อมูลได้อย่างสะดวกสบาย
