Famous game that was built with Java.

เกมดังที่สร้างด้วย Java

Minecraft ถูกสร้างขึ้นโดยใช้ภาษาโปรแกรม Java พร้อมกับ Lightweight Java Game Library (LWJGL) นอกจากนี้ Minecraft ยังมีการพอร์ตเวอร์ชันอื่นๆ ไปยังภาษาโปรแกรมและแพลตฟอร์มอื่น ๆ เช่น C++ สำหรับ Windows 10 Edition และ Bedrock Edition และ JavaScript สำหรับ Minecraft: Education Edition แต่เวอร์ชันดั้งเดิมที่ใช้ภาษา Java ยังคงเป็นเวอร์ชันที่ได้รับความนิยมที่สุด
