What is TypeScript?

TypeScript คืออะไร

ไทป์สคริปต์ (TypeScript) เป็นภาษาโปรแกรมระดับสูงแบบโอเพ่นซอร์สฟรีที่พัฒนาและดูแลโดย Microsoft มันเป็น syntactic superset ของ JavaScript โดยเพิ่มการพิมพ์แบบ static โดยใช้ syntax เหมือนกันทั้งหมด มีจุดประสงค์เพื่อลดจำนวน bug ที่จะเกิดขึ้นและเตือนให้แก้ตั้งแต่ตอนเขียน ในขณะที่ JavaScript แบบปกติจะพบได้ก็ต่อเมื่อนำไป run แล้ว

Why use TypeScript over Vanilla JavaScript?

ทำไมถึงควรใช้ Typescript แทน JavaScript ปกติ

ข้อดี
1. แก้บัคง่ายขึ้น
2. การเขียนที่เร็วขึ้น
ข้อเสีย
1. ต้อง setup เพิ่ม
2. ต้องเรียนรู้เพิ่มเติมจาก JavaScript อีกที
